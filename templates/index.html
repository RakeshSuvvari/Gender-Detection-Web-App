Gender Detection using Face Detection


In this Project gender of person is detected using face detection. To achieve this python libraries like Tensorflow,CV2,Keras,Numpy,Cvlib, etc are used. And the Web App is built using HTML, CSS, Bootstrap and the backend is developed using Flask.


Click on the Start button to open Camera

The gender of a person will be shown around the Face with some Accurarcy, in the Camera Display.
